About this project

Read N Ride is my final project for the Web Development Immersive program at General Assembly. The app gives District of Columbia commuters hand-picked long-form journalism pieces to read during their Metro ride. The app is powered by the WMATA API and a REST API that I built to interface with Longreads.com's article data.

Read N Ride is my first full-fledged application using the MeteorJS framework, which I learned independently over the course of a week. View Read N Ride in action here and the source code here. Have a suggestion? Submit an issue!
