About this project

I created this game with three classmates as a collaborative project for the Web Development Immersive class. Users can test their Twitter trivia knowledge by answering questions about popular users. These questions are randomly generated using data from the Twitter REST API.

My role in this project was initially back-end work, ranging from database modeling and migrations to custom controller actions. I managed the team Trello board and shared scrum master duties with another group member. We worked together to refactor all of our JavaScript code to be object-oriented and also set up a Heroku background worker to query the API at regular intervals.

To practice SCSS, I re-styled the app on my own as a secondary project. The original game is deployed here and the updated version here.
