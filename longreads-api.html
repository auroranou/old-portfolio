About this project

As part of my Web Development Immersive final project, I built this REST API to serve Longreads.com data in JSON format. The API relies on a MongoDB collection that is populated using a Node.js data scraper.

This project is my first foray into using Express.js + Mongoose. Documentation for the API is available here and the source code may be forked here.

Specs

  • + Node.js
  • + MongoDB 2.0
  • + Cheerio 0.18
  • + Express 4.12
  • + Mongoose 3.8.23
